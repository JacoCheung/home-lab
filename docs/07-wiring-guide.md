# 布线与安装指南

> 精装修房屋的智能设备安装位置与注意事项

---

## 一、总体原则

- **精装修已有零线**：所有墙壁开关位直接替换为 Aqara H1 Pro 零火版
- **不需要额外布线**：Zigbee 无线通信，传感器/无线开关贴墙即可
- **窗帘电机需预留电源**：检查窗帘盒内是否有电源插座，没有需电工拉线
- **弱电箱/电视柜附近**：放置 M3 中枢和 Mini PC，需有网线和电源

## 二、逐设备安装说明

### 2.1 M3 方舟中枢

| 项目 | 说明 |
|------|------|
| 位置 | 客厅电视柜旁或弱电箱附近 |
| 连接 | 网线连路由器（推荐有线），USB-C 供电 |
| 注意 | 不要放金属柜子里（影响 Zigbee/IR 信号） |
| 红外覆盖 | M3 自带红外，对准空调/电视方向放置 |

### 2.2 智能开关（替换现有开关）

| 区域 | 型号 | 接线 | 注意 |
|------|------|------|------|
| 玄关 | H1 Pro 单键 | L(火)+N(零)+灯线 | 关闭总闸后操作 |
| 走廊 | H1 Pro 单键 | 同上 | |
| 客厅 | H1 Pro 三键 ×2 | L+N+3路灯线 | 每个三键控制3组灯 |
| 主卧 | H1 Pro 双键 ×2 | L+N+2路灯线 | 门口+床头各一个 |
| 次卧 | H1 Pro 双键 | L+N+2路灯线 | |
| 多功能室 | H1 Pro 双键 | L+N+2路灯线 | |
| 厨房 | H1 Pro 双键 | L+N+2路灯线 | 主灯+操作灯 |
| 主卫 | H1 Pro 单键 | L+N+灯线 | |
| 客卫 | H1 Pro 单键 | L+N+灯线 | |

**替换步骤**:

1. 关闭对应回路断路器
2. 拆下原面板，拍照记录接线
3. 火线(L)接 L 端子，零线(N)接 N 端子，灯线接 L1/L2/L3
4. 装入底盒，上好面板
5. 合闸，Aqara App 扫码配对

### 2.3 筒射灯（T2/T3）

| 项目 | 说明 |
|------|------|
| 安装方式 | 替换原有吊顶筒灯，标准开孔 75mm |
| 接线 | 火线+零线，通过 Zigbee 控制亮度色温 |
| 分组 | 同一房间按区域分 2-3 组，配合多键开关 |

> 如果原有筒灯开孔不是 75mm，需购买对应的转换环

### 2.4 LED 灯带 + 调光模块

| 项目 | 说明 |
|------|------|
| 灯带位置 | 吊顶灯槽内，沿灯槽走线 |
| 调光模块位置 | 吊顶内或开关底盒内 |
| 接线 | 220V -> 调光模块 -> 灯带驱动 -> LED灯带 |
| 分段 | 每条灯带配一个调光模块 |

### 2.5 窗帘电机 C3

| 位置 | 电源检查 | 轨道长度 |
|------|----------|----------|
| 阳台左窗 | 窗帘盒内需有插座 | ~3m |
| 阳台右窗 | 同上 | ~3m |
| 客厅西窗 | 同上 | ~2.9m |
| 主卧北窗 | 窗帘盒内需有插座 | ~1.5m |
| 主卧东窗 | 同上 | ~1.5m |
| 次卧北窗 | 同上 | ~2.3m |
| 多功能室窗×2 | 视改造后 | 按实际 |
| 多功能室床帘 | 墨菲床上方 | ~2m |

**安装步骤**:

1. 确认窗帘盒内有电源（没有需请电工拉线）
2. 安装窗帘轨道（轨道商家一般提供安装）
3. C3 电机挂在轨道一端
4. 插电，Aqara App 扫码配对
5. 手动拉到两端标定行程

### 2.7 FP2 区域设置要点

FP2 需要在 Aqara App 中划分检测区域:

- **玄关 FP2**: "入口区域"
- **客厅 FP2**: "沙发区" + "餐桌区" + "阳台区"
- **主卧 FP2**: "床区" + "衣柜区"
- **多功能室 FP2**: "书桌区" + "床区"

## 三、网络规划

```
光猫 (桥接模式)
   │
   ▼
UniFi Cloud Gateway Ultra (拨号+路由)
   │
   ├── 有线 ─── Mini PC (HA)
   ├── 有线 ─── M3 中枢
   ├── 有线 ─── G3 摄像头（如果走得到线）
   └── Wi-Fi ─── 其他设备
```

- Mini PC 和 M3 强烈建议有线连接
- 所有 Zigbee 设备通过 M3 通信，不占用 Wi-Fi 带宽
- Wi-Fi 设备: FP2、G3、G4、晾衣机、扫地机、青萍

---

## 四、安全提醒

1. **更换开关前必须关闭对应断路器**，用电笔验证无电
2. 窗帘电机电源线如果不够长，**请专业电工接线**
3. PZEM-004T 涉及 220V，**必须由电工安装 CT 线圈**
4. 所有 3M 胶粘的传感器，表面需先用酒精擦拭除油
5. ESP32 等低压设备可自行安装，注意防水防潮
