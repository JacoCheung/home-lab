小米 Miloco

Xiaomi Miloco · 视觉 + 自然语言规则

以米家摄像机为视觉源、端侧大模型做视频理解，用自然语言设家庭规则，与 HA 方案并存

GitHub: XiaoMi/xiaomi-miloco

📍

在本方案中的定位

现有方案Miloco 补充
Home Assistant：自动化、设备控制、3D 仪表盘自然语言设规则、对话式控制
Ollama + 小智：语音、AI 对话端侧视觉理解（看家场景、视频问答）
Frigate：AI 视频分析大模型级视频理解
Aqara / 米家：设备执行米家设备/场景检索与执行、米家通知

融合方式：与 HA 并行。HA 管 Aqara 与自动化，Miloco 管「摄像头视觉 + 自然语言规则 + 米家」。

🖥️

硬件要求（Miloco 服务器）

  • CPU：x64
  • 显卡：NVIDIA 30 系及以上，显存 ≥8GB（建议 12GB+）
  • 存储：≥16GB 可用（本地模型）
  • 系统：Linux（建议 Ubuntu 22.04+）或 Windows 10+ WSL2；macOS 暂不支持

当前 Mini PC N100 无独显，需单独一台带 NVIDIA GPU 的主机，或与 Frigate 共用一台 GPU 服务器。

一键安装

满足硬件与 Docker 环境后，在 Linux 或 WSL2 中执行：

bash 
bash -c "$(wget -qO- https://xiaomi-miloco.cnbj1.mi-fds.com/xiaomi-miloco/install.sh)"

或克隆仓库后执行 bash scripts/install.sh。

详细步骤见官方 Docker 部署与使用文档。
